## Haiku Minigame
Interactive haiku completion game powered by AI that generates poems with one missing line. Users fill in the blank, then receive AI-generated feedback with a 1-10 rating and personalized commentary on their creative contribution. Project made with the intention to better understand AI and it's usage.

## Live Demo
https://haiku-minigame.vercel.app/

> **Note:** Initial load may take 30-60 seconds as the backend spins up (free hosting tier).

## Tech Stack

### Frontend
- **React** - UI framework
- **TypeScript** - Type safety
- **Tailwind CSS** - Styling
- **React Query** - Data fetching & caching
- **Vite** - Build tool

### Backend
- **Express.js** - Web framework
- **Node.js** - Runtime environment
- **Google Gemini API** - AI integration

### Deployment
- **Frontend**: Vercel
- **Backend**: Render

## Lessons learned:
- Better understanding of how to manage AI and its prompts, as well as how to validate the responses received.
- Enhanced my knowledge around react-query by digging deeper into its features, such as manual refetching.
- Improved my skills with custom hooks and context, further understanding the use cases of these tools
